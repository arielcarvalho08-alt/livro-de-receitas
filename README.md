# 📖 Livro de Receitas Compartilhado

Este é um projeto focado em reunir as melhores receitas culinárias desenvolvidas de forma colaborativa via Git e GitHub.

## 🍽️ Lista de Receitas
* [Receitas Principais](receitas.txt) - Veja a lista completa de receitas adicionadas pela equipe.

## 🤝 Como Contribuir
1. Abra uma **Issue** relatando qual receita falta.
2. Crie uma **Branch** com o nome da receita.
3. Adicione a receita no arquivo `receitas.txt`.
4. Faça o commit usando `Fixes #numero_da_issue`.
5. Abra um **Pull Request** para revisão!
